Installation
============

``reciprocalspaceship`` is currently only available on GitHub. For installation, we recommend
cloning the repository and installing the package to a ``conda`` environment::

  git clone https://github.com/Hekstra-Lab/reciprocalspaceship
  cd reciprocalspaceship
  python setup.py install

Notes:

- These installation instructions will be updated when ``reciprocalspaceship`` is available
  through PyPI and can be installed with ``pip`` or ``conda``
